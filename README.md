# AniBenchDB

A benchmark dataset for music structure analysis, focused on Japanese anime songs (anisong) with complex and irregular phrase structures.

## Motivation

- Irregular phrase lengths (non-multiples of 8 bars)
- Mid-song time signature changes
- Frequent key changes
- Complex fill patterns with mixed time signatures

AniBenchDB is designed to target real-world DJ use cases where precise phrase boundary detection matters.

## Audio

Audio files are **not included**. Tracks can be matched by `title` + `artist` fields.
